# Todoアプリ 仕様書

## 1. 概要

シンプルで軽量なTodoアプリ。タスクの追加・編集・削除・検索・フィルタリングを提供する。
現フェーズはローカル保存のみ。将来的にクラウド同期・認証に対応できる設計とする。

## 4. 機能要件

### 4.1 タスク管理

| 機能 | 内容 |
|---|---|
| タスク追加 | タイトル・期限日・優先度・タグ・ステータスを設定して作成 |
| タスク編集 | 各項目を後から変更可能 |
| タスク削除 | 個別削除（確認ダイアログあり） |

### 4.2 タスクの属性

| 属性 | 型 | 詳細 |
|---|---|---|
| タイトル | テキスト | 必須 |
| 期限日 | 日付（任意） | 未設定可 |
| 優先度 | 選択肢 | 後述「4.3 優先度」参照 |
| タグ | 複数選択（自由入力） | 後述「4.5 タグ」参照 |
| ステータス | 選択肢 | 後述「4.4 ステータス」参照 |
| 作成日時 | 日時 | 自動設定 |
| 更新日時 | 日時 | 自動更新 |

### 4.3 優先度

デフォルトで以下の2種類を提供。ユーザーが任意に追加・編集・削除可能。

| 優先度名 | 説明 |
|---|---|
| 絶対やる | 必ず対応が必要なタスク |
| 余裕があったらやる | 時間があれば対応するタスク |

### 4.4 ステータス

デフォルトで以下の3種類を提供。ユーザーが任意に追加・編集・削除可能。

| ステータス名 |
|---|
| 未着手 |
| 進行中 |
| 完了 |

### 4.5 タグ

- 自由入力で作成（テキスト）
- 1つのタスクに複数のタグを付与可能
- タグ管理画面でタグの編集・削除が可能
- タグを削除した場合、そのタグを持つタスクからも除去される

### 4.6 一覧表示

- タスクを一覧表示する
- デフォルトの表示順: 作成日時の降順（新しい順）
- タスクカードに表示する情報: タイトル、期限日、優先度、ステータス、タグ
- 「完了」ステータスのタスクはデフォルトで非表示（フィルタで表示切り替え可能）

### 4.7 フィルタリング

以下の条件で絞り込み可能。複数条件の組み合わせに対応。

| フィルタ項目 | 詳細 |
|---|---|
| タグ | 選択したタグを持つタスクのみ表示 |
| 優先度 | 選択した優先度のタスクのみ表示 |
| ステータス | 選択したステータスのタスクのみ表示 |
| 期限日 | 今日・今週・今月・期限切れ などの範囲指定 |

### 4.8 ソート

以下の項目でソート可能（昇順・降順の切り替えあり）。

| ソート項目 |
|---|
| 作成日時（デフォルト）|
| 更新日時 |
| 期限日 |
| 優先度 |
| タイトル（五十音順）|

### 4.9 テキスト検索

- タイトルを対象にキーワード検索
- フィルタリングと組み合わせて使用可能

## 5. データモデル

### tasks テーブル

| カラム名 | 型 | 説明 |
|---|---|---|
| id | INTEGER (PK) | 自動採番 |
| title | TEXT | タイトル（必須）|
| due_date | DATE (nullable) | 期限日 |
| priority_id | INTEGER (FK) | 優先度 |
| status_id | INTEGER (FK) | ステータス |
| created_at | DATETIME | 作成日時 |
| updated_at | DATETIME | 更新日時 |

### priorities テーブル

| カラム名 | 型 | 説明 |
|---|---|---|
| id | INTEGER (PK) | 自動採番 |
| name | TEXT | 優先度名 |
| sort_order | INTEGER | 表示順 |
| is_default | BOOLEAN | アプリ初期データかどうか（削除可能）|

### statuses テーブル

| カラム名 | 型 | 説明 |
|---|---|---|
| id | INTEGER (PK) | 自動採番 |
| name | TEXT | ステータス名 |
| sort_order | INTEGER | 表示順 |
| is_default | BOOLEAN | アプリ初期データかどうか（削除可能）|

### tags テーブル

| カラム名 | 型 | 説明 |
|---|---|---|
| id | INTEGER (PK) | 自動採番 |
| name | TEXT | タグ名（一意）|

### task_tags テーブル（中間テーブル）

| カラム名 | 型 | 説明 |
|---|---|---|
| task_id | INTEGER (FK) | タスクID |
| tag_id | INTEGER (FK) | タグID |

